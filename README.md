# Lyme-disease-locations- 
A Python-based tool that uses the News API and public news to identify and map U.S. states with reported Lyme disease activity. This web app is locally hosted through Streamlit  


## Built with 
- Python
- News Api
- Beautiful Soup
- Streamlit
- Json
